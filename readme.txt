== Description ==

A Shocking Red Publish Button for your post page.

This plugin simple use CSS to change the Publish button in the post page to Red. So you dont publish a post accidentally because you push the wrong button.


== Installation ==

1. Upload the full directory into your /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. DONE!
